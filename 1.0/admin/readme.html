Справка


Тип контента "страницы" и "чанки" отличаются друг от друга тем, что чанки можно вставлять в код страницы или шаблона в любом месте любое количество раз.
При этом в "страницы" и "чанки" можно добавлять любой текст или html-код.
"Чанки" можно использовать, например, для создания меню в шаблоне. Для этого необходимо создать "чанк" с html-кодом меню и вставить его в соответсвующее место в своём шаблоне. "Чанки" добавляются конструкцией вида: chunk_номерID
Этот номер можно посмотреть в панели администрирования рядом с именем соответствующего "чанка".

Создание собственного шаблона


Для создания собственного шаблона необходимо сначала в папке templates создать новую папку с произвольным именем. Имя новой папки должно содержать только символы латиницы и цифры и начинаться с буквы. Это имя нужно вставлять в поле "Шаблон" при создании страниц.
После этого необходимо скопировать из templates/blank все файлы в созданную папку.
Можно создавать произвольный шаблон, основная задача заключается в том, чтобы так распределить тэги по файлам header.php и footer.php, чтобы начальный и завершающий тэг места, куда вы планируете вставлять контент "страниц" были разнесены по разным файлам.
Например в файле header.php будет: ...начальный код...<div>, а в файле footer.php соответственно <⁄div>...конечный код...
Вставка своего кода в php файлы должна соответствовать правилам языка PHP. То есть, необходимо в файле header.php после строчки . '<body>'; на новой строке вставить свой код в одинарных кавычках, добавив в начале точку и пробел, а в конце точку с запятой. После <body>' точку с запятой ОБЯЗАТЕЛЬНО убрать!
В файле footer.php всё немного наоборот. Там нужно свой код в одинарных кавычках вставлять после знака =, но до '<⁄body>', при этом '<⁄body>' надо сместить на одну строчку вниз, а перед ним поставить точку и пробел.

Для того, чтобы движок автоматически подставлял относительные пути в ссылки на скрипты, таблицы стилей, картинки, можно использовать конструкцию вида:
href="⁄templates⁄'.$row_template[0].'⁄...
в файлах header.php и footer.php своего шаблона.
Также необходимо учитывать то, что если вы хотите использовать разные шаблоны для разных страниц, вы должны их создать отдельно каждый и при создании страницы указывать в поле "Шаблон" имя нужного шаблона.
